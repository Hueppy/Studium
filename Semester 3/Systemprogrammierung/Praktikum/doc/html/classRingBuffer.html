template<typename T, std::size_t size, typename alloc_t = std::allocator<T>>
class RingBuffer< T, size, alloc_t >

Template class to represent a ring buffer

Parameters
Tis the type of elements to be stored (via pointer of type T*)
sizeis the max. number of elements that can be stored
alloc_tis the type of the allocator used to allocate the pointers to the elements (optional)

Reading the first (oldest) element

If an element has been read, the head points to the next element and count is decremented. The read element is not released.

Returns
Returns a pointer to the first (i.e. oldest) element of the buffer, but does not (yet) delete it; returns nullptr if buffer is empty

Adding a new element

Appends the new element to the end of the queue. If the buffer is full, the oldest element will be deleted with an allocator of type alloc_t and the pointer to the new element will be inserted in this location.

Parameters
datais a pointer to the element to be added (allocated with an allocator of type alloc_t)
